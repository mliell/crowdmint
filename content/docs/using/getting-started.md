# Getting Started

Welcome to CrowdMint! This guide will help you understand the basics of our decentralized crowdfunding platform and get you started in minutes.
 
## Key Features

- **Transparent**: All campaigns and transactions are recorded on-chain
- **Secure**: Built with smart contracts for trustless interactions
- **Global**: No borders, no banks, acessible to anyone with a Web3 wallet and USDC.

## Set Up Your Wallet

Before you can interact with CrowdMint, you'll need a Web3 wallet:

1. **Download a Wallet**: Install Rabby, MetaMask, Coinbase Wallet, or another Web3 wallet
2. **Create an Account**: Follow the wallet's setup instructions
3. **Add USDC**: Get USDC from faucet here: [Easy Faucet Arc Testnet](https://easyfaucetarc.xyz/) or [Testnet Faucet](https://faucet.circle.com/)
4. **Connect to CrowdMint**: Click "Connect Wallet" on our platform

## Start Using CrowdMint

Once connected, you can:

- **Create a Campaign**: Share your project idea with the community
- **Back a Campaign**: Support projects you believe in
- **Manage Donations**: Track your contributions and rewards
